## 3. 회고
### 김경민
- 텍스트 데이터의 불균형을 해소하는 방법을 고민하는 과정에서 Back Translation이나 Easy Data Augmentation과 같은 테크닉이 있음을 새롭게 알게 되었다.
- 특히 텍스트 데이터 다중분류 문제에 어떻게 접근하고 해결할 수 있는지 배울 수 있는 시간이었다.
- 중간중간 어려움을 겪었는데 함께 논의하며 성공적으로 프로젝트를 마무리한 것 같아서 뿌듯하다. 

### 박지현
- 두 달 동안 많이 배웠다. NLP를 다뤄볼 수 있어 좋았던 시간이었다!

### 조민주 
- 자연어 처리 과정에 대해 공부할 수 있는 매우 유익한 프로젝트였다. 특히 LSTM을 튜닝하는 여러가지 방법을 시도해볼 수 있어 즐거웠다.

### 조승연
- 웹크롤링, 텍스트 데이터 증강 등에 대한 공부와 학기 중 세션 내용 복습을 통해 많은 것을 배워갈 수 있는 프로젝트였다.
